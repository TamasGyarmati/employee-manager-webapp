Employee Manager

Employee Manager is a simple frontend interface that allows users to manage employee data. Users can add new employees, edit existing records, or delete them as needed. The application communicates with a backend API built using ASP.NET Core (C#) and utilizes Entity Framework Core to interact with a Microsoft SQL Server database. All operations (create, read, update, delete) are handled via HTTP API calls, ensuring real-time data synchronization between the UI and the database.
