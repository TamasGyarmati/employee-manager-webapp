Student Handler

Student Handler is a simple frontend interface that allows users to manage student data. Users can add new students, edit existing records, or delete them as needed. The application communicates with a backend API built using ASP.NET Core (C#) and utilizes Entity Framework Core to interact with a Microsoft SQL Server database. All operations (create, read, update, delete) are handled via HTTP API calls, ensuring real-time data synchronization between the UI and the database.
